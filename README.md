# Internship Management System (Backend API)

[![Laravel Version](https://img.shields.io/badge/Laravel-v11.x-FF2D20?logo=laravel)](https://laravel.com)
[![PHP Version](https://img.shields.io/badge/PHP-v8.2%20%2F%20v8.3-777BB4?logo=php)](https://www.php.net)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Aplikasi Backend berbasis RESTful API untuk mengelola data magang/prakerin (Praktik Kerja Industri) siswa. Sistem ini mencakup manajemen pengguna (Siswa, Guru Pembimbing, Admin), pencatatan absensi berbasis koordinat (Geolokasi), pengajuan jurnal harian, serta monitoring tempat magang.

---

## 🛠️ Spesifikasi Teknologi

* **Framework:** Laravel 13.8
* **Bahasa Pemrograman:** PHP >= 8.3
* **Database:** MySQL 
* **Autentikasi:** Laravel Sanctum
* **Arsitektur:** RESTful API (JSON Response)
